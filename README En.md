## Definition

Master Editor is a unified modding tool for Resident Evil 4 Ultimate HD Edition. It combines multiple tools and editors into a single software with a clean, organized interface, allowing you to modify most of the game systems without directly using hex editors like HxD.

The tool is built for RE4 modders to provide an easier and faster workflow for editing game files and the executable (`bio4.exe`).

---

# Modules

- Code Manager
- OSD Editor
- CNS Editor

- AEV Options Editor
- MDT Color Editor

- Weapons & Items Editor *(Not available in source code)*

# Module Breakdown

## Code Manager

A module featuring over 180 pre-built codes within a single interface.

Features:
- Enable or disable codes.
- Easily modify values.
- Apply modifications directly to `bio4.exe`.
- Execute codes in standard mode or Real-Time.
- Utilize codes without the need for HxD.

Codes are categorized into sections for streamlined access.

---

## OSD Editor

This module is dedicated to editing OSD files.

File functionality covers systems such as:
- Removing items from the inventory.
- Triggering AEV events.
- Linking in-game events.

All edits are handled through a clean interface tailored to the file structure, removing the need for manual hex editing.

---

## CNS Editor

This module allows you to modify the internal limits of the game, including:
- Enemy count.
- Effect count.
- Model count.
- Other internal engine limits.

---

## SND Editor

This module is currently under development.

---

## AEV Options Editor

This module is designed for easily editing AEV OPTION messages.

Features:
- Open TXT files.
- Link messages with AEV.
- Modify text-specific codes.
- Preview text rendering directly as it appears in-game.

Familiarity with the AEV OPTION system makes this module's functionality immediately clear.

---

## MDT Color Editor

Features:
- Modify MDT colors.
- Create custom colors.
- Real-time preview of text presentation in-game.
- Easily edit message colors.

This module mirrors the live-preview concept found in the AEV OPTION editor.

---

## Room Init Editor

This module enables you to transfer the properties/coding of any stage to another with a single click.

Example:
- Transfer properties from `R103` to `R107`.

---

## Scripts

A dedicated module for running helper scripts and utility tools associated with game modification.

---

## Lock AEV with Key

A module built to manage files that link AEV triggers to keys.

Includes:
- AVL Editor
- Event.cfg Editor

Allows direct file modification via a graphical user interface without resorting to HxD.

---

## Weapons & Items Editor

An extensive module for modifying weapon systems and in-game items.

Modifiable attributes:
- Weapon firepower.
- Upgrade count limits.
- Upgrade costs.
- Merchant items inventory.
- Upgrades available at the Merchant.
- Item prices.
- Item stock limits.
- Item types.
- Item size dimensions within the inventory grid.
- Combining items to create new ones.
- Starting items.
- Starting health and currency values.
- Numerous other parameters.

All managed within a simple, structured interface.

---

# Key Features

- Clean and simple interface.
- Full Arabic and English language support.
- All utility tools integrated into a single executable.
- Minimizes the need for HxD in most modification processes.
- Supports direct modification of game files.
- Supports instant, real-time modification.
- Over 180 pre-built codes ready in the Code Manager.
- GitHub-integrated update checking system.
- Built-in backup system for sensitive operations.
- Supports multiple unique file formats specific to Resident Evil 4.
- Live preview for text and colors across multiple modules.
- Winlator support.
- Linux support via Wine.
- Native Windows support.
- No manual library installation required when using the compiled version.
- Unified module layout for quick navigation between tools.
